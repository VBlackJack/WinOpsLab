# Reseau

!!! abstract "Objectifs du module"

    A la fin de ce module, vous serez capable de :

    - [ ] Configurer l'adressage IPv4/IPv6 et les sous-reseaux
    - [ ] Gerer le pare-feu Windows avec regles avancees
    - [ ] Deployer des solutions d'acces distant (VPN, NPS)
    - [ ] Configurer NIC Teaming et IPAM
